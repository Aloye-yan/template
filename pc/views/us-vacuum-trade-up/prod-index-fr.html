Les questions que vous vous posez peut-être

Dois-je renvoyer mon ancien appareil pour obtenir le bon d'achat ?
+
Non, vous pouvez garder votre ancien appareil.
Où puis-je acheter un PURE ONE STATION avec le code de réduction fourni ?
+
Le code de réduction peut être utilisé pour l'achat d'un PURE ONE STATION sur Amazon ou sur notre boutique officielle.
Dois-je forcément avoir un appareil Tineco pour participer au programme de remplacement ?
+
Non, le programme de remplacement est ouvert aux aspirateurs de toutes les marques.
Combien vais-je recevoir pour mon appareil ?
+
Le bon d'achat que pouvez recevoir pour votre appareil dépend du prix d'achat de votre ancien aspirateur. Si son prix est inférieur à 350€ (350€ inclus), vous recevrez un bon d'achat de 100€. Si son prix dépasse 350€, vous recevrez un bon d'achat de 160€. Veuillez noter que le prix déduit sur la commande peut varier à hauteur de 5 € maximum en fonction de la boutique.
En combien de temps vais-je recevoir mon bon d'achat ?
+
Nous avons besoin de 3 à 5 jours ouvrés pour vérifier vos informations et procéder à l'envoi de votre bon d'achat en fonction de la valeur de votre ancien appareil. Gardez un œil sur votre messagerie !
Y a-t-il une date limite pour l'utilisation du bon d'achat ?
+
Oui. Le programme de remplacement se tient du 1er Décembre au 10 Décembre 2023, et le code de réduction est valable du 1er au 17 Décembre 2023. Il est impératif d'utiliser le code pendant cette période.
Comment puis-je utiliser le code de réduction ?
+
Les codes peuvent être utilisés uniquement lors du passage d'une commande. En savoir plus : https://www.amazon.com/gp/help/customer/display.html%3FnodeId%3DGPAU9BCECRXQBFQR
